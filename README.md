# C-Monye

Приложение для операций над
криптовалютой: средствами и счетами.
А также перевод средств со счета на счет

## Установка и запуск приложения

`npm install`
`cd ./c-money`

### Запуск

`npm run start`
приложение доступно на (http://localhost:3000)

### Сборка текущего билда

`npm run build`
